## Agent skills

### Output language

All artifacts produced by mattpocock/skills should be written in Chinese. Keep critical English terms unchanged when they are labels, workflow states, commands, template keys, file names, code identifiers, or established skill vocabulary. See `docs/agents/output-language.md`.

### Issue tracker

Issues and PRDs are tracked in GitHub Issues using the `gh` CLI. See `docs/agents/issue-tracker.md`.

### Triage labels

Use the default mattpocock/skills triage labels. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context repo: read root `CONTEXT.md` and `docs/adr/` when present. See `docs/agents/domain.md`.
